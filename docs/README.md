## 🔄 Updating Documentation

### When to Update

| Change Type | Update These Docs |
|-------------|-------------------|
| New feature | FEATURES.md, README.md |
| Bug fix | CHANGELOG.md (if exists) |
| API change | DATA_*.md, PROVIDERS.md |
| UI change | UI_*.md, screenshots |
| Dependency update | README.md (tech stack) |

### How to Update

1. **Make the change** in code
2. **Update related docs** immediately
3. **Cross-reference** other docs
4. **Test** any commands you changed
5. **Commit** docs with code
